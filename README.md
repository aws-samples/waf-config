# Description
Demo project to show the use of cdk to deploy waf configurations.

# Deploy
this can easily be deployed via cloudshell

1. git clone git clone https://github.com/aws-samples/waf-config
2. cd waf-config
3. npm install
4. cdk bootstrap
5. cdk deploy 

# Update CDK
sudo npm install -g aws-cdk
npm install
cdk bootstrap
cdk deploy
